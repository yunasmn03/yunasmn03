que sont GET et POST ?

Ce sont les deux plus grandes requêtes de HTTP

quelques mots utiles :

- GET est la méthode la plus courante pour demander une ressource. - POST est employé lorsque l'on envoie au serveur les données issues d'un formulaire - DELETE permet de supprimer une ressource sur le serveur - PUT permet de modifier une ressource sur le serveur

le codes de statut de réponse HTTP 202 indique la reussite d'une requête Le code 404 signifie que la page est introuvable
